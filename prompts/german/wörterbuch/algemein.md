INPUT = Eine Liste an Eingabeparameter die du erwarten kannst;
OUTPUT = Eine Liste an erwarteten Ausgabeparametern welche du in deiner Antwort ausgeben musst;
ANTWORT = Die Ausgabe deiner OUTPUT Werte;
BEISPIEL = Ein oder mehrere Beispiel für was du erwarten kannst und beachten musst;
FORMAT = Das Format welches deine ANTWORT einhalten muss, BEISPIEL:(JSON Formatierung, Antworte in einem Satz);
AUFGABE = Deine Aufgabe, mit welcher du dich auschließlich beschäftigst, beschreibt den gebenen OUTPUT, die ANTWORT im rahmen von FORMAT;

MITARBEITENDEN = Eine Gruppe an Professionellen Agenten welche unterschiedliche Aufgaben und Spezialisierungen haben mit welchen du immer zusammenarbest;
USER = Ein Mensch welcher mit den MITARBEITENDEN interagiert um ein Problem zu lösen.
FRAGEBOGEN = Ein Fragebogen welcher Fragen zu einer Aufgabe enthält und zur Kommunikation zwischen den MITARBEITENDEN verwendet wird;
ANTWORTBOGEN = Ein Antworbogen auf dem Fragen eines FRAGEBOGEN, Frage für Frage, beantwortet werden. Enthällt die selbe anzahl an Antworten wie Fragen auf dem FRAGEBOGEN; 
KONVERSATION = Eine List an Fragen und Antworten welche zwischen einem MITARBEITENDEN und einem USER geführt wird;
